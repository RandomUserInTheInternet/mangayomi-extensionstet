**Key implementation details** (verified from multiple existing Kotlin/Java scrapers of haho.moe from Aniyomi, CloudStream, Saikou):

- **Search URL**: `https://haho.moe/anime?q=QUERY&page=N`
- **Popular URL**: `https://haho.moe/anime?s=vdy-d&page=N` (views descending)
- **Latest URL**: `https://haho.moe/anime?s=rel-d&page=N` (release date descending)
- **Anime list selector**: `ul.anime-loop.loop > li > a`
- **Title selector**: `div span.thumb-title` or `div.label > span`
- **Thumbnail**: `img` inside the `<a>` tag
- **Critical cookie**: `loop-view=thumb` must be sent with all requests to enable thumbnail display
- **Detail page**: title = `li.breadcrumb-item.active`, cover = `img.cover-image.img-thumbnail`, description = `div.card-body`, genres = `li.genre span.value`, status = `li.status span.value`
- **Episode list selector**: `ul.episode-loop > li > a` with `div.episode-number`/`div.episode-slug` for ep number and `div.episode-title`/`div.episode-label` for ep title
- **Pagination**: `ul.pagination li.page-item a[rel=next]`
- **Video extraction**: Episode page has an `<iframe>` → fetch iframe → get `video#player > source` tags. Also supports server dropdown: `ul.dropdown-menu > li > a.dropdown-item` → extract `?v=` param → fetch `/embed?v=ID` → get `video#player > source`
- **Video sources** are direct `.mp4` files hosted on domains like `s1.filegasm.com` with `<source>` tags having `src` and `title` (quality like "720p", "1080p") attributes
- **isNsfw**: true (the site is "Hentai AHO Streaming")
